WHATTHEFLOP REST API Documentation

WTF API

Create

  • description: Create a new user
  • request: POST /signup/
    • content-type: application/json
    • body: {"username":"admin", "password":"pass4admin"}"
      • (String) username
      • (String) password
  • response: 200
    • body: "admin signed up."
  • response: 500
    • body: server error
  • response: 400
    • body: Password is missing
    • body: Username is missing
  • response: 401
    • body: admin username already exists
    • body: The username is in an invalid format! Only alphanumeric characters!
  • response: 409
    • body: admin username already exists.
$ curl -X POST
        -H "Content-Type: application/json"
        -d '{"username":"admin", "password":"pass4admin"}'
        https://www.whattheflop.me/signup/

Create

  • description: Create a Suggestion
  • request: POST /room/<room_id>/suggestion/
    • content-type: application/json
    • body:
      • (String) suggestion - suggestion name
      • (Int) userId
      • (String) suggestionJson - suggestion object json
  • response: 200
    • body: suggestion: new suggestion
  • response: 500
    • body: server error
  • response: 400
    • body: The username is missing
    • body: The suggestions is missing
  • response: 409
    • body: Suggestion already exists
$ curl -X POST
      -H "Content-Type: application/json"
       -d '{"suggestion":"suggestion", "userId":"3681310", "suggestionJson":"{suggestion: suggestionJson}"}'
       https://www.whattheflop.me/room/1/suggestion/
